# Gym-website

# Oxygen Gym - وبسایت باشگاه بدنسازی فرضی



یک وبسایت تک‌صفحه‌ای (Landing Page) حرفه‌ای و ریسپانسیو برای باشگاه بدنسازی با طراحی مدرن، انیمیشن‌های نرم و تجربه کاربری عالی.

## پیش‌نمایش زنده
[پیش‌نمایش زنده](https://elyasahmadzadeh.github.io/Gym-website/) 

## ویژگی‌های پروژه

- طراحی کاملاً **ریسپانسیو** (مناسب برای موبایل، تبلت و دسکتاپ)
- استفاده از **Bootstrap 4.5.3** برای ساختار و کامپوننت‌ها
- ناوبری ثابت (Fixed Navbar) با انیمیشن تغییر ارتفاع هنگام اسکرول
- هدر تمام‌صفحه با تصویر پس‌زمینه و انیمیشن ورود متن
- بخش‌های مختلف: افتخارات، خدمات، گالری تصاویر با تب، اسلایدر مربیان، پلن‌های عضویت، فرم تماس
- فرم ثبت‌نام/تماس با طراحی خاص و clip-path
- فوتر کامل با لینک‌ها و آیکون‌های شبکه‌های اجتماعی
- پشتیبانی از جهت **RTL** (راست‌به‌چپ) برای زبان فارسی
- فونت‌های سفارشی ایرانی (IranYekan) و انگلیسی (Roboto)
- انیمیشن‌های نرم با CSS و جاوااسکریپت خالص

## تکنولوژی‌های استفاده‌شده

- HTML5
- CSS3 (با انیمیشن و متغیرهای سفارشی)
- JavaScript (vanilla) برای تعاملات دینامیک
- Bootstrap 4.5.3
- Font Awesome 7
